# vue-parcel
[parcel-bundler] で vue.jsを動かすサンプル

## Requirement
* npm or yarn

## Installation
```shell
$ git clone https://github.com/sasatake/vue-parcel.git
$ cd vue-parcel
$ yarn install
```

## Usage
```shell
$ yarn dev
```


[parcel-bundler]: https://github.com/parcel-bundler/parcel
[vue.js]: https://jp.vuejs.org/index.html
